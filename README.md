## Purpose:
To enable a user to visually compare their resume against a job description

## Example:

Source – e.g. Resume: A B
Target – e.g. Job Description: B C D

A = Yellow
B = Green
C = Red
D = Grey

                          Job Description Relevant
                            No                  Yes
          Resume  Yes   Caution = Yellow    Good = Green
                  No    N/A = Grey          Bad = Red
